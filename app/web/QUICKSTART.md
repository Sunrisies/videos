# 快速开始指南

## 新功能概览

本次更新为视频播放应用添加了以下功能：

### 1. 🔐 密码验证
- 访问视频库前需要输入密码
- 默认密码：`video123`
- 会话期间保持登录状态

### 2. 📱 移动端交互优化
- **单击视频画面**：显示/隐藏控制栏
- **拖动视频画面**：调整播放进度（左右滑动）
- **进度预览**：拖动时显示时间预览气泡

---

## 立即体验

### 步骤 1：启动开发服务器

```bash
cd app/web
npm run dev
```

服务器将在 http://localhost:13000 启动

### 步骤 2：访问应用

打开浏览器访问：http://localhost:13000

### 步骤 3：输入密码

首次访问会看到密码验证页面，输入默认密码：`video123`

### 步骤 4：体验新功能

进入视频列表，点击任意视频开始播放：

1. **试试单击视频画面**：控制栏会显示/隐藏
2. **试试拖动视频画面**：在屏幕上左右滑动调整进度
3. **观察预览气泡**：拖动时会显示时间预览

---

## 自定义配置

### 修改密码

编辑文件：`app/web/.env.local`

```env
NEXT_PUBLIC_APP_PASSWORD=你的新密码
```

重启开发服务器使修改生效。

### 禁用密码验证

注释掉密码配置：

```env
# NEXT_PUBLIC_APP_PASSWORD=video123
```

---

## 移动端测试

推荐使用以下方式测试移动端功能：

### 方法 1：浏览器开发者工具

1. 按 F12 打开开发者工具
2. 点击设备模拟器图标（Toggle device toolbar）
3. 选择移动设备（如 iPhone、iPad、Pixel）
4. 刷新页面开始测试

### 方法 2：真实移动设备

1. 确保手机和电脑在同一局域网
2. 在手机浏览器访问：`http://[你的电脑IP]:13000`
3. 输入密码后开始体验

---

## 功能说明

### 密码验证

- ✅ 首次访问需要输入密码
- ✅ 刷新页面无需重新输入
- ✅ 关闭浏览器后需重新验证
- ✅ 支持回车键提交

### 单击切换控制栏

- ✅ 单击视频画面显示控制栏
- ✅ 再次单击隐藏控制栏
- ✅ 播放中3秒无操作自动隐藏
- ✅ 暂停时控制栏保持显示

### 拖动调整进度

- ✅ 在视频画面上左右滑动
- ✅ 向右滑动：快进
- ✅ 向左滑动：快退
- ✅ 拖动时显示时间预览
- ✅ 松开手指应用进度变化

---

## 常见问题

### Q: 拖动功能不工作？
A: 拖动功能需要触摸屏或触摸板支持。PC 端鼠标请使用进度条调整进度。

### Q: 如何清除登录状态？
A: 在浏览器控制台执行：
```javascript
sessionStorage.removeItem('authenticated')
location.reload()
```

### Q: 忘记密码怎么办？
A: 查看 `app/web/.env.local` 文件中的 `NEXT_PUBLIC_APP_PASSWORD` 配置。

### Q: 如何在生产环境使用？
A: 
1. 创建 `.env.production` 文件并配置密码
2. 运行 `npm run build` 构建
3. 运行 `npm start` 启动生产服务器
